Experiment & Testing Page


 - Get User Details button is used to demonstrate how user details can be used after a user has logged in
- Logout button can be used to logout the user, to test it works, try the Get User Details button afterwards, if the logout was successful it will return null
- Get Session Data is used to retrieve session data, this is one way we could share data between pages of the tool. Try clicking a skin tone on the skin tone page and then clicking this button to see such concept in action
